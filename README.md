# Geotropism Simulator 🌱

<p align="center">
  <img src="repo-banner/assets/banner.png" alt="Geotropism Simulator Banner" width="100%">
</p>

A clean, interactive educational web simulation modeling plant root gravitropism (geotropism). Explore how plant roots perceive and respond to gravity, the role of auxin distribution, and how tilting affects root growth direction.

---

## ✨ Features

* **Interactive Tilt:** Tilt the pot at various angles to observe real-time directional changes in root growth.
* **Auxin View (Magnifier):** Toggle the magnifier view to visualize hormone (auxin) concentrations accumulating on the lower side of the root during gravitropic bending.
* **Gravity Simulation:** Toggle and simulate vector gravitational forces acting on the root cap statoliths.
* **Step-by-Step Growth:** Watch the simulation unfold progressively using the play/grow controls.
* **Reset Functionality:** Instantly reset the simulation to test different angles and hypotheses.
